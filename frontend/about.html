Project guide

How to use SearchX

SearchX demonstrates a custom search engine built with Node, Express, MongoDB, Bootstrap, and vanilla JavaScript.

Search user workflow

  1. Open the Search page.
  2. Type a keyword such as mongodb, ranking, or api.
  3. Submit the search form.
  4. Use category, source, tag, and date filters to narrow results.
  5. Open a result to view the full record detail page.

Admin workflow

  1. Use Content Admin to create, update, delete, and re-index records.
  2. Use Search Admin to create and activate ranking profiles.
  3. Review metrics for corpus size, index terms, top tags, and search activity.

How search works

The backend tokenizes each published record, removes stopwords, and writes custom inverted index entries to MongoDB. Search requests look up matching terms, fetch candidate records, apply filters, and score results using the active ranking profile.
